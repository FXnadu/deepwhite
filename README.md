# DeepWhite Theme for Typora

极简、优雅、专注阅读的 Typora 主题

## 特色功能

- **圆形任务列表**：Mac 风格的绿色圆形复选框
- **标题自动编号**：从 H2 开始的多级编号
- **霞鹜文楷字体**：优雅的中文排版，内置字体文件
- **暖白色背景**：护眼的阅读体验
- **GitHub 警告框**：支持多种类型的提示框
- **代码高亮**：精美的代码块样式
- **响应式设计**：适配不同屏幕尺寸

## 安装方法

### 方法一：完整安装（推荐）

1. 下载整个项目文件夹
2. 将 `deepwhite.css` 和 `deepwhite-fonts` 文件夹复制到 Typora 主题目录
3. 重启 Typora 并选择 DeepWhite 主题

### 方法二：仅主题文件

1. 下载 `deepwhite.css` 文件
2. 复制到 Typora 主题文件夹
3. 重启 Typora 并选择 DeepWhite 主题

> **注意**：方法二将使用系统默认字体，可能无法获得最佳视觉效果

详细安装说明请参考：[INSTALL.md](INSTALL.md)
